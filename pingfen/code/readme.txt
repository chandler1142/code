使用：
    1. 将程序放置在合适位置，以~/code为例。
    2. 打开terminal
    3. cd ~/code/
    4. sudo chmod 755 ./*.*
    6. ./score -p [excel文件路径]
注意:
    1. 使用前请备份当前的数据，以免程序执行出错损坏文件
    2. 运行程序时候，最好关闭excel
    3. 腾讯的播放量取的单位是(万)，有自定义增长请手动修改
条件：
    1. 执行前，保证excel当天的单元格存在。比如2月9日的单元格要存在。sheet1中爱奇艺的趋势的当天单元格也要存在
    2. 保证电脑是联网状态
已有功能：
    sheet1:
        1. 爱奇艺 数据+评分+排名
        2. 腾讯 数据+评分+排名
        3. 芒果 数据+评分+排名
        4. PPTV 数据+评分+排名
    sheet2:
        1. 自动填充程序执行时刻的芒果数据到当天列
    sheet3:
        1. 自动填充程序执行时刻的PPTV数据到当天列
